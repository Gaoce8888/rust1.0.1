常见问题

如何重置客服密码？

+

可以通过以下方式重置密码：

  1. 联系系统管理员重置
  2. 使用忘记密码功能（如果已配置）
  3. 直接修改数据库中的密码字段

系统支持哪些文件格式？

+

系统支持以下文件格式：

  • 图片：JPG、PNG、GIF、WebP
  • 文档：PDF、DOC、DOCX、TXT
  • 压缩包：ZIP、RAR
  • 其他：MP3、MP4（小文件）

如何备份系统数据？

+

建议定期备份以下数据：

  • 数据库文件（SQLite/Sled）
  • 配置文件
  • 上传的文件
  • 日志文件

可以使用系统提供的备份工具或手动备份。

系统性能如何优化？

+

可以通过以下方式优化系统性能：

  • 配置Redis缓存
  • 优化数据库查询
  • 调整并发连接数
  • 使用CDN加速静态资源
  • 定期清理日志和临时文件
